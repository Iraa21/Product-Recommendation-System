
# ML-Based Product Recommendation System

## Overview

This project is a content-based product recommendation system designed to provide personalized product suggestions for e-commerce platforms. The system analyzes product attributes using Natural Language Processing (NLP) techniques and recommends similar products based on user preferences and product characteristics.

The recommendation engine utilizes TF-IDF vectorization and Cosine Similarity to identify relevant products and generate real-time recommendations through a Flask-based web application.

---

## Features

* Content-based product recommendation
* TF-IDF vectorization for feature extraction
* Cosine similarity-based matching
* Real-time recommendation generation
* Interactive web interface using Flask
* MySQL database integration
* Product similarity search and ranking
* Data visualization and exploratory analysis

## System Workflow

1. Product data is collected and stored in the database.
2. Product features are preprocessed and cleaned.
3. TF-IDF Vectorization converts textual information into numerical feature vectors.
4. Cosine Similarity calculates similarity scores between products.
5. The recommendation engine identifies the most relevant products.
6. Flask serves recommendations through a user-friendly web interface.
